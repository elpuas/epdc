El *Sistema de Gestión de Fonogramas y Sencillos* fue desarrollado como una aplicación sobre WordPress para modelar y administrar datos estructurados del sector musical. Más que un sitio web, se diseñó una arquitectura que automatiza relaciones, notificaciones y flujos administrativos complejos.

        ## El Desafío

        La agencia requería una solución capaz de:

        - Registrar fonogramas y sencillos con múltiples autores y ejecutantes.
        - Modelar relaciones dinámicas entre obras y personas involucradas.
        - Gestionar estructuras complejas con repeaters anidados.
        - Automatizar notificaciones según rol (autor, ejecutante, administrador).
        - Generar exportaciones estructuradas compatibles con procesos internos.
        - Permitir importaciones masivas sin comprometer integridad de datos.

        No se trataba de crear contenido simple, sino de diseñar un flujo robusto de gestión de información crítica.

        ## Nuestra Solución

        Como parte de un proyecto *white label*, desarrollamos una arquitectura personalizada basada en:

        - Custom Post Types (`fonograma` y `sencillo`).
        - Modelado avanzado con ACF y repeaters estructurados.
        - Gravity Forms + Advanced Post Creation.
        - Integración vía REST API para consultar autores dinámicamente.
        - Inserción automática de autores dentro de repeaters de Gravity Forms mediante JavaScript.
        - Registro automático de relaciones entre obra y participantes.
        - Sistema de notificaciones segmentadas por rol.
        - Plugin personalizado para exportaciones.
        - Extensión del sistema de exportación nativo de WordPress para adaptar el formato a necesidades específicas.

        Además, se desarrolló un importador custom que permite cargas estructuradas manteniendo consistencia en las relaciones entre datos.

        El resultado fue una aplicación interna estable, donde WordPress funciona como motor de datos y no solo como gestor de contenido.
