# We Build Real Tools for Real Problems

At ElPuas Digital Crafts, we spend a large part of our time creating custom solutions for our clients. Very often, the problem is not solved by installing more generic plugins, but by truly understanding how the business works and building specific tools around that.

One of the most recent examples is EPDC Conversations, a modern CTA and conversation system for WordPress focused on WhatsApp conversions.

The project started after identifying something very common across many websites in Latin America: WhatsApp became the main sales and support channel, but most available solutions were heavy, hard to customize, and offered almost no useful metrics about real conversions.

Instead of depending on external widgets, we decided to build our own solution.

EPDC Conversations includes:

- modern floating buttons
- dynamic Gutenberg blocks
- reusable shortcodes
- Google Analytics 4 integration
- local conversion tracking
- internal metrics dashboard
- performance-optimized architecture
- server-side rendering
- accessibility and responsive design

But most importantly, we can now better understand how users interact with each site:

- which pages generate the most conversations
- which campaigns convert better
- what kind of traffic is actually working
- which CTA performs best

All of this information is recorded in a lightweight and privacy-aware way, making it possible to have real metrics without depending only on external platforms.

And this is not the only internal project we have developed.

Across different projects, we have built specific tools to solve very concrete client needs, for example:

- Puro Confort Product Pricing  
  Lightweight system to display product configurations and prices without the full complexity of WooCommerce.

- EPDC Product Selector  
  Dynamic product selector focused on fast experiences that are easy for clients to manage.

- GreenPack Core  
  Modular WooCommerce controls built specifically for the GreenPack platform.

- PBCR Agent Mode  
  System that enables a “broker mode” to share properties through unique, minimalist, no-branding URLs.

- PBCR Similar Properties Filter  
  Custom similar-properties filter based on deep hierarchical locations.

- Premium Brokers – Agent Form CC  
  RealHomes extension that adds automatic CC behavior to agent forms.

- Smart Home Leads  
  Integration between RealHomes forms and the Smart Home API for centralized lead handling.

- AgentPress  
  AI-powered chatbot for real estate agents using APIs and automation for real estate and ecommerce.

We have also worked on:

- smart hub systems
- automated platform-to-platform connections
- Agent Mode workflows
- internal tools for lead handling and automation
- custom WordPress blocks
- optimized dashboards and editorial flows

Many of these projects start first as internal solutions and then evolve into reusable tools that we can adapt to different clients and scenarios.

In addition, much of this work is developed under an open source philosophy. We like building maintainable, transparent, and reusable tools, not black boxes that are impossible to extend.

Because in the end, for us, the goal is not to add complexity.

It is to create tools that genuinely help businesses work better.
