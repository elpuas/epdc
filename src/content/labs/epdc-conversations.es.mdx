# Construimos herramientas reales para problemas reales

En ElPuas Digital Crafts pasamos gran parte del tiempo creando soluciones personalizadas para nuestros clientes. Muchas veces el problema no se resuelve instalando más plugins genéricos, sino entendiendo realmente cómo funciona el negocio y construyendo herramientas específicas alrededor de eso.

Uno de los ejemplos más recientes es EPDC Conversations, un sistema moderno de CTAs y conversaciones para WordPress enfocado en conversiones vía WhatsApp.

El proyecto nació después de detectar algo muy común en muchos sitios de Latinoamérica: WhatsApp se convirtió en el principal canal de ventas y soporte, pero la mayoría de soluciones disponibles eran pesadas, difíciles de personalizar y prácticamente no ofrecían métricas útiles sobre conversiones reales.

En lugar de depender de widgets externos, decidimos crear nuestra propia solución.

EPDC Conversations incluye:

- botones flotantes modernos
- bloques dinámicos para Gutenberg
- shortcodes reutilizables
- integración con Google Analytics 4
- tracking local de conversiones
- dashboard interno de métricas
- arquitectura optimizada para performance
- renderizado server-side
- accesibilidad y responsive design

Pero lo más importante es que ahora podemos entender mejor cómo interactúan los usuarios con cada sitio:

- qué páginas generan más conversaciones
- qué campañas convierten mejor
- qué tipo de tráfico está funcionando
- qué CTA tiene mejor rendimiento

Toda esta información se registra de forma ligera y respetando privacidad, permitiendo tener métricas reales sin depender únicamente de plataformas externas.

Y este no es el único proyecto interno que hemos desarrollado.

A lo largo de distintos proyectos hemos construido herramientas específicas para resolver necesidades muy concretas de nuestros clientes, por ejemplo:

- Puro Confort Product Pricing  
  Sistema ligero para mostrar configuraciones y precios de productos sin la complejidad completa de WooCommerce.

- EPDC Product Selector  
  Selector dinámico de productos enfocado en experiencias rápidas y fáciles de administrar para clientes.

- GreenPack Core  
  Controles modulares para WooCommerce desarrollados específicamente para la plataforma GreenPack.

- Modo Agente PBCR  
  Sistema que habilita un “modo corredor” para compartir propiedades mediante URLs únicas, minimalistas y sin branding.

- PBCR Similar Properties Filter  
  Filtro personalizado de propiedades similares basado en ubicaciones jerárquicas profundas.

- Premium Brokers – Agent Form CC  
  Extensión para RealHomes que agrega CC automático en formularios de agentes.

- Smart Home Leads  
  Integración entre formularios de RealHomes y la API de Smart Home para manejo centralizado de leads.

- AgentPress  
  Chatbot para agentes inmobiliarios impulsado por IA utilizando APIs y automatización para real estate y ecommerce.

También hemos trabajado en:

- sistemas de hubs inteligentes
- conexiones automatizadas entre plataformas
- Agent Mode workflows
- herramientas internas para manejo de leads y automatización
- bloques personalizados para WordPress
- dashboards y flujos editoriales optimizados

Muchos de estos proyectos nacen primero como soluciones internas y luego evolucionan a herramientas reutilizables que podemos adaptar a distintos clientes y escenarios.

Además, gran parte de este trabajo se desarrolla bajo una filosofía open source. Nos gusta construir herramientas mantenibles, transparentes y reutilizables, no cajas negras imposibles de extender.

Porque al final, para nosotros, el objetivo no es agregar complejidad.

Es crear herramientas que realmente ayuden a que los negocios funcionen mejor.
