# Grid System

Our grid system is built using modern CSS features like CSS Grid and Container Queries, providing a flexible and responsive layout solution.

## Features

- 12-column grid system
- Responsive breakpoints (sm, md, lg, xl)
- Container query support
- Semantic HTML structure
- Accessible by default

## Usage

### Basic Grid

```astro
<div class={styles.grid}>
  <div class="col-span-6">Half width</div>
  <div class="col-span-6">Half width</div>
</div>
```

### Responsive Grid

```astro
<div class={styles.grid}>
  <div class="col-span-12 md:col-span-6 lg:col-span-4">
    Full width on mobile, half on medium, third on large
  </div>
</div>
```

### Nested Grids

```astro
<div class={styles.grid}>
  <div class="col-span-6">
    <div class={styles.grid}>
      <div class="col-span-6">Nested half</div>
      <div class="col-span-6">Nested half</div>
    </div>
  </div>
</div>
```

## Breakpoints

- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px

## Best Practices

1. Always use semantic HTML elements
2. Include proper heading hierarchy
3. Use container queries for component-level responsiveness
4. Maintain consistent spacing using CSS variables
5. Test across all breakpoints
